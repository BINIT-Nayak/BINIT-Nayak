## Featured Projects

### [Sneaky](https://github.com/BINIT-Nayak/Sneaky) — Swipe-based product discovery

A full-stack shopping discovery experience with swipe interactions, wishlist and cart flows, merchant links, product analytics, and personalized recommendations.

- Built a responsive **React + TypeScript** frontend with Redux Toolkit
- Developed secure **Java 21 + Spring Boot** APIs with JWT authentication
- Added **PostgreSQL**, **Redis** caching, and optional **Kafka** activity events
- Designed a hybrid recommendation pipeline with rule-based candidate generation, ML probability scoring, diversity reranking, graceful fallback, and cached results

Repositories: [Frontend](https://github.com/BINIT-Nayak/Sneaky) · [Backend](https://github.com/BINIT-Nayak/Sneaky_Backend) · [ML Reranker](https://github.com/BINIT-Nayak/Sneaky_Recommender)

### [VibeJournal](https://github.com/BINIT-Nayak/VibeJournal) — Private mood journal

A privacy-conscious journaling app for mood check-ins, deeper reflection, CBT-inspired insights, mood trends, context tracking, and music suggestions. Built with **Next.js, TypeScript, Prisma, and PostgreSQL**.

### [Snapgram](https://github.com/BINIT-Nayak/Snapgram) — Social media application

A responsive social platform with authentication, profiles, posts, image uploads, likes, saves, and search. Built with **React, TypeScript, Appwrite, TanStack Query, React Hook Form, and Zod**.

### [YouTube Reimagined](https://github.com/BINIT-Nayak/YouTube-Reimagned) — Video discovery UI

A personalized video browsing experience featuring search, recommendations, continuous playback, and a responsive interface powered by **React, TypeScript, Redux, Tailwind CSS, and the YouTube API**.
